# 灵构 (GuideMMPromptCraft)
引导式多模态提示词生成AI | 视频/图片/文案 专业提示词生成

## 功能
- 3轮智能引导提问，精准收集需求
- 支持视频/图片/文案 三类提示词
- 会话管理、暗黑模式、语音播报
- 智谱GLM-4驱动，专业提示词输出

## 启动
1. 安装依赖：pip install -r requirements.txt
2. 配置 .env：ZHIPU_API_KEY=你的密钥
3. 运行：python ai_main.py
